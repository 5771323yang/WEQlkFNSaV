# 船舶监造管理系统 java575

## 项目概述

船舶监造管理系统是基于Springboot和Vue的前后端分离项目，采用Javaee技术栈。该项目遵循MVC三层体系结构，结合Spring、SpringBoot、Mybatis和Vue等框架进行开发，使用Maven进行项目管理。系统使用MySQL数据库作为数据存储平台，实现了一套基于B/S架构的Web应用系统。

## 技术栈

- **前端**：Vue.js
- **后端**：Springboot、Javaee、Mybatis

- **项目管理**：Maven
